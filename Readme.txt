

基础函数放置在lib 比如数据库连接函数，字符串函数等


核心函数放置在core  比如对商品操作，管理用户操作的相关函数  一般来说没有硬性要求要放置在那个文件下，只是为了更好的找到相应函数，当然代码编辑器里面通过查找可以方便的找到函数定义


生成的函数文件，文件配置文件要“注册”到 include.php 文件  其他文件夹包含的时候只需要包含 include.php 就好，避免了复杂的文件包含步骤



对数据库进行建表，增删查改表属性，记得及时共享相关  sql操作语句   


配置文件是config.php


所有css  js  images放置在style 文件夹下  


每次进行github 的sync的时候，记得尽量在description里面写详尽代码修改  当然欢迎运用github其他高大上的功能

建议每完成一部分就提交一次


admin下面是对后台管理的操作

根目录下是前端用户页面设计
